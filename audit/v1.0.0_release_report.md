**One-paragraph release announcement:**

> Aksu v1.0.0-alpha is a neural Turkish morphological atomizer: given any Turkish word or sentence, it decomposes it into root + ordered suffix tags (e.g., `ev +Noun +POSS.3PL +ABL`) using a frozen BERTurk + 1M-parameter reranker, with no GPU required for training. The package is a clean rename of the prior `kokturk` library — compat shims preserve existing `from kokturk import` code through v1.x. The 98.3% em_argmax (candidate-index accuracy, 5-seed ensemble) figure is the only performance claim verified in this release; em_string and all inference throughput figures are pending Orfoz/akya-cuda SLURM runs. The TR-Gold-Morph v1 corpus (80,537 manually annotated Turkish tokens, CC BY 4.0) is the dataset backbone.
